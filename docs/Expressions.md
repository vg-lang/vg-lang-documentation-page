# Expressions in VG Language

Expressions form the building blocks of VG Language, including:
- Arithmetic operations (`+`, `-`, `*`, `/`, `%`)
- Logical operations (`&&`, `||`, `!`)
- Comparisons (`==`, `!=`, `<`, `>`, `<=`, `>=`)

Example:
```vg
var result = (10 + 5) * 2;
print(result);  ## Output: 30
```
